# leo_core

Standalone Law Enforcement Core for FiveM using ox_lib.

## Features
- LEO interaction menu (ox_lib)
- Cuff / escort / vehicle interactions
- Duty & department system
- Panic button
- ShotSpotter / gunshot detection
- Duty blips
- Configurable & localized
- Standalone (no framework)

## Requirements
- ox_lib

## Installation
1. Place `leo_core` in your resources folder
2. Ensure ox_lib
3. Add to server.cfg:
   ensure ox_lib
   ensure leo_core
4. Configure `config.lua`
5. Add ACE permissions

## Permissions
